## 九、关键词配置

### 编辑关键词配置
```powershell
notepad search_information\TrendRadar\config\frequency_words.txt
```

### 关键词语法
```
[WORD_GROUPS]          - 词组定义区域
华为                   - 简单关键词
/正则表达式/           - 正则匹配
+必须词                - 必须包含
!排除词                - 排除包含
=> 别名               - 显示别名
[组名]                 - 词组标签
@5                     - 限制条数
```
