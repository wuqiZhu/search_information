# 日常精简命令速查

最后更新：2026-05-21

所有命令在 PowerShell 中执行，假设当前目录为项目根目录。

---

## 一、启动服务

### TrendRadar（信号采集）
```powershell
cd search_information\TrendRadar
python main.py
```

### n8n（Docker 自动化）
```powershell
cd search_information\n8n
docker-compose up -d
```

### analyse_information（RSS + AI 分析）
```powershell
cd analyse_information
python analyzer/pipeline.py --rss
```

### analyse_information（每日摘要）
```powershell
cd analyse_information
python analyzer/pipeline.py --digest
```

---

## 二、停止服务

### 停止 TrendRadar
```powershell
taskkill /F /IM python.exe
```

### 停止 n8n
```powershell
cd search_information\n8n
docker-compose down
```

---

## 三、查看状态

### 查看 Docker 容器
```powershell
docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"
```

### 查看 TrendRadar 日志（最新 50 行）
```powershell
Get-Content search_information\TrendRadar\logs\trendradar.log -Tail 50
```

### 查看 analyse_information 日志（最新 50 行）
```powershell
Get-Content analyse_information\shared\logs\scheduler.log -Tail 50
```

### 查看 Python 进程
```powershell
Get-Process python -ErrorAction SilentlyContinue | Select-Object Id, StartTime, CPU | Format-Table
```

## 四、测试验证

### 测试钉钉通知
```powershell
cd search_information\TrendRadar
python -c "import hmac,hashlib,base64,urllib.parse,time,requests,os;from dotenv import load_dotenv;load_dotenv('../.env');w=os.environ['DINGTALK_WEBHOOK'];s=os.environ['DINGTALK_SECRET'];ts=str(round(time.time()*1000));ss=f'{ts}\n{s}';sg=urllib.parse.quote_plus(base64.b64encode(hmac.new(s.encode(),ss.encode(),hashlib.sha256).digest()));p={'msgtype':'markdown','markdown':{'title':'TrendRadar 测试通知','text':'## 测试通知\n\n配置正确！'}};r=requests.post(f'{w}&timestamp={ts}&sign={sg}',json=p,timeout=10);print(r.text)"
```

### 测试 RSS 源连通性
```powershell
cd search_information\TrendRadar
python -c "import feedparser;f=feedparser.parse('https://www.phoronix.com/rss.php');print(f'Phoronix: {len(f.entries)} entries')"
```

### 测试 DeepSeek API
```powershell
cd analyse_information
python -c "from analyzer.config import config;print(f'API: {config.ai.base_url}')"
```

## 五、清理重置

### 清理 TrendRadar 已处理信号（重新匹配）
```powershell
taskkill /F /IM python.exe
Remove-Item search_information\TrendRadar\data\processed_signals.json -Force
Remove-Item search_information\TrendRadar\data\signals\* -Force
cd search_information\TrendRadar
python main.py
```

### 重启 n8n
```powershell
cd search_information\n8n
docker-compose down
docker-compose up -d
```

## 六、配置文件位置速查

| 文件 | 路径 | 用途 |
|------|------|------|
| 环境变量 | `search_information/.env` | 所有密钥（钉钉、邮箱、API） |
| TrendRadar 配置 | `search_information/TrendRadar/config.yaml` | 信号源、关键词、通知 |
| n8n 配置 | `search_information/n8n/.env` | n8n 环境变量 |
| 分析器配置 | `analyse_information/analyzer/config.yaml` | RSS源、关键词、分类 |
| 分析器环境 | `analyse_information/.env` | 分析项目密钥 |
