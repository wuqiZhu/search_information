# 日常精简命令速查

最后更新：2026-05-25

所有命令在 PowerShell 中执行，假设当前目录为项目根目录。

---

## 一、启动服务

### TrendRadar（信息采集）
```powershell
cd search_information\TrendRadar
python main.py
```

### n8n（Docker 自动化，可选）
```powershell
cd search_information\n8n
docker-compose up -d
```

### analyse_information（RSS + AI 分析）
```powershell
cd analyse_information
python analyzer/pipeline.py --rss
```

### analyse_information（每日摘要）
```powershell
cd analyse_information
python analyzer/pipeline.py --digest
```

### analyse_information（每周摘要）
```powershell
cd analyse_information
python analyzer/pipeline.py --weekly-digest
```

### analyse_information（分析单个URL）
```powershell
cd analyse_information
python analyzer/pipeline.py --url "https://example.com/article"
```

### find_job（求职岗位抓取）
```powershell
cd find_job
python -m job_scraper.main
```

---

## 二、停止服务

### 停止 TrendRadar
```powershell
taskkill /F /IM python.exe
```

### 停止 n8n
```powershell
cd search_information\n8n
docker-compose down
```

## 三、查看状态

### 查看 Docker 容器
```powershell
docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"
```

### 查看 TrendRadar 日志（最新 50 行）
```powershell
Get-Content search_information\TrendRadar\logs\trendradar.log -Tail 50
```

### 查看 analyse_information 日志（最新 50 行）
```powershell
Get-Content analyse_information\shared\logs\scheduler.log -Tail 50
```

### 查看 find_job 日志（最新 50 行）
```powershell
Get-Content find_job\data\logs\scraper.log -Tail 50
```

### 查看 Python 进程
```powershell
Get-Process python -ErrorAction SilentlyContinue | Select-Object Id, StartTime, CPU | Format-Table
```

### 查看 find_job 反馈统计
```powershell
cd find_job
python -c "from job_scraper.feedback import get_feedback_stats; import json; print(json.dumps(get_feedback_stats(), indent=2))"
```

## 四、测试验证

### 测试 TrendRadar 钉钉通知
```powershell
cd search_information\TrendRadar
python -c "import hmac,hashlib,base64,urllib.parse,time,requests,os;from dotenv import load_dotenv;load_dotenv('../.env');w=os.environ['DINGTALK_WEBHOOK'];s=os.environ['DINGTALK_SECRET'];ts=str(round(time.time()*1000));ss=f'{ts}\n{s}';sg=urllib.parse.quote_plus(base64.b64encode(hmac.new(s.encode(),ss.encode(),hashlib.sha256).digest()));p={'msgtype':'markdown','markdown':{'title':'TrendRadar 测试通知','text':'## 测试通知\n\n配置正确！'}};r=requests.post(f'{w}&timestamp={ts}&sign={sg}',json=p,timeout=10);print(r.text)"
```

## 五、清理重置

### 清理 TrendRadar 已处理信号（重新匹配）
```powershell
taskkill /F /IM python.exe
Remove-Item search_information\TrendRadar\data\processed_signals.json -Force
Remove-Item search_information\TrendRadar\data\signals\* -Force
cd search_information\TrendRadar
python main.py
```

### 清理 find_job 已见岗位（重新抓取）
```powershell
Remove-Item find_job\data\seen_jobs.json -Force
cd find_job
python -m job_scraper.main
```

### 重启 n8n
```powershell
cd search_information\n8n
docker-compose down
docker-compose up -d
```

### 更新 Boss Cookie
```powershell
cd find_job
python scripts/get_cookie.py
```

## 六、配置文件位置速查

| 文件 | 路径 | 用途 |
|------|------|------|
| 环境变量 | `search_information/.env` | TrendRadar 密钥（钉钉、邮箱、API） |
| TrendRadar 主配置 | `search_information/TrendRadar/config/config.yaml` | 热榜平台、RSS源、通知、AI分析 |
| TrendRadar 关注词 | `search_information/TrendRadar/config/frequency_words.txt` | 关注词（支持正则/必须词/排除词） |
| n8n 配置 | `search_information/n8n/.env` | n8n 环境变量 |
| 分析器配置 | `analyse_information/analyzer/config.yaml` | RSS源、AI模型、分类规则 |
| 分析器环境 | `analyse_information/.env` | DeepSeek API Key |
| 通知中心配置 | `analyse_information/shared/notification_config.yaml` | 钉钉 Webhook、优先级规则 |
| 求职简历配置 | `find_job/profile.json` | 教育背景、技能、评分标准、黑名单 |
| 求职环境变量 | `find_job/.env` | BOSS_COOKIE、DEEPSEEK_API_KEY、DINGTALK_WEBHOOK |
| 搜索查询列表 | `find_job/job_scraper/constants.py` | 47个搜索查询和阈值常量 |
