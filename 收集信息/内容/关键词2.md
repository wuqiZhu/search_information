## 🏔️ 外部信息获取金字塔（脚本专用版）

> 帮你找到实习/工作的直接入口。

| 关键词          | 说明                           |
| :-------------- | :----------------------------- |
| 嵌入式Linux实习 | 主力岗位                       |
| C++开发实习     | 辅助岗位                       |
| 2026校招提前批  | 暑期实习+秋招提前批            |
| 招聘网站爬取    | Boss直聘、实习僧、猎聘、牛客网 |

### 第2层：面经与题库（直接应对面试）

> 了解别人被问过什么，针对性准备。

| 关键词                     | 说明          |
| :------------------------- | :------------ |
| 嵌入式面经2026             | 重点关注      |

| 算法题高频（LeetCode热题） | 只抓简单/中等 |

### 第3层：政策与福利（省钱、拿补贴）

> 学生身份能带来实在的好处，很多人不知道。

| 关键词                        | 说明                             |
| :---------------------------- | :------------------------------- |
| 应届生租房补贴                | 深圳1250/月、杭州1万/年等        |
| 学生云服务器优惠              | 阿里云、腾讯云、华为云学生套餐   |

| 应届生人才引进                | 各地对本科生的落户/租房/生活补贴 |

### 第4层：行业动态（了解外界在发生什么）

> 防止闭门造车，把握技术趋势和公司动向。

| 关键词          | 说明                           |
| :-------------- | :----------------------------- |
| 嵌入式行业新闻  | 芯片、物联网、汽车电子等       |
| Linux内核新特性 | 版本更新、新驱动模型等         |
| 目标公司动态    | 博众精工、大疆、汇顶、联发科等 |
| 开源社区热点    | GitHub trending（嵌入式标签）  |

### 第5层：竞赛与项目（破圈机会）

> 帮你找到能写在简历上的亮点。

| 关键词         | 说明                                 |
| :------------- | :----------------------------------- |
| 嵌入式竞赛2026 | 如嵌入式芯片设计竞赛、物联网设计竞赛 |

## 📌 脚本如何配合这个金字塔？

你可以为每一层设置独立的抓取任务：

| 层级  | 抓取内容                       | 输出形式                        | 提醒条件                 |
| :---- | :----------------------------- | :------------------------------ | :----------------------- |
| 第1层 | 新发布的嵌入式实习岗位         | 表格（公司+岗位+链接+发布日期） | 每日汇总一次             |
| 第2层 | 新发布的面经帖子               | 提取公司、岗位、被问到的技术点  | 当日推送（不堆积）       |
| 第3层 | 新发布的租房补贴、学生优惠     | 政策名称+截止日期+申请链接      | 截止前15天、7天、1天提醒 |
| 第4层 | 目标公司新闻、GitHub热门项目   | 标题+摘要+来源链接              | 每周汇总一次             |
| 第5层 | 新发布的竞赛通知、开源之夏项目 | 竞赛名称+报名时间+官网          | 报名开始前提醒           |
